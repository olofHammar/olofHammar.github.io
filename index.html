CHEFNOTES

Om appen: Chefnotes är din personliga kokbok i telefonen. Du kan skapa recept genom appens mallar eller genom att scanna ett recept från en bild. Alla recept du skapar sparas i din personliga bok.
Du kan även bläddra och inspireras av andra användares recept. Dessa recept kan du spara som favoriter i din egna bok så att du enkelt kommer åt dom senare.

Språk: SwiftUI
Databas: Firebase

TIC TAC TOE

Om appen: Ett tre i rad-spel som du kan spela med en kompis eller med appens inbyggda AI-spelare.
Spelet håller koll på varje spelares namn och vinststatistik.

Språk: UIKit

NEIGHBOURS

Om appen: Neighbours är en chat-app som vill uppmuntra grannsämja. Med Neighbours kan du chatta med personer i din närhet (1-10 kilometer) och även se om någon av dessa personer behöver hjälp med någonting.

Språk: Kotlin
Databas: Firebase

FLAGQUIZ

Om appen: I denna app testas dina flaggkunskper.
Genom att svara på tio, tjugo eller trettio frågor så utmanas du att försöka ta dig in på topplistan.

Språk: Kotlin
Databas: SQL
